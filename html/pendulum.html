Inverted Pendulum Project

For my Dynamic Modeling course, ME313, we built a controlled, fixed-axis, inverted pendulum. The pivot of the pendulum is attached to a gear box with a 5V motor providing torque. As part of the project we learned to represent the physical system in terms of state variables, as well as describing it using laplace transfer functions. At the same time I was also in a more advanced Control Systems course, ME/ECE481, which allowed me to use more advanced control practices in my inverted pendulum.

To initially stabilize the system I designed a PID (proportional, integral, derivative) controller. This algorithm can be found on my Github page here. I also used tools such as the Runge-Kutta method for solving differntial equations, in both Matlab and python, in order to model how different control methods would effect the system.
